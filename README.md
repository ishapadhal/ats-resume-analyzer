# ATS Resume Analyzer

An AI-powered ATS tracking system that evaluates resumes against job descriptions using LLaMA 3 via Groq API.

## Features
- Resume PDF upload
- Job description input
- Match percentage calculation
- Missing keywords detection
- AI feedback

## Tech Stack
- Python
- Streamlit
- Groq API (LLaMA 3)
- PyPDF2

## Setup

1. Clone repo
2. Install dependencies
3. Add .env with GROQ_API_KEY
4. Run streamlit run app.py
